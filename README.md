# Superstar Cities Thesis

Replication code and data for my thesis on German superstar cities and housing markets.

## Structure

- `data/raw/` : original datasets
- `data/processed/` : cleaned data
- `R/` : scripts
- `output/` : tables and figures

## Replication

Run scripts in order:

1. 01_prepare_greix_data.R
2. 02_prepare_income_data.R
3. 05_regressions_and_figures.R
