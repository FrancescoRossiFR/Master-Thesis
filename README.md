# Superstar Cities and Housing Supply Inelasticity in Germany

This repository contains the data pipeline, econometric analysis, and visualization scripts for my Master's Thesis. The study investigates how housing supply constraints drive price divergence across German cities using the "Superstar Cities" framework.

## Project Structure

- **R/**: Contains all R scripts, including the master execution file.
- **data/raw/**: Original datasets (GREIX index, Eurostat income, WWII destruction data).
- **data/processed/**: Cleaned and merged panels generated by the pipeline.
- **output/tables/**: LaTeX-ready regression tables.
- **output/figures/**: All thesis visualizations (PDF format).
- **Master-Thesis.Rproj**: The RProject configuration file.

## How to Reproduce the Analysis

This project is designed for one-click reproducibility. Follow these steps:

1. **Open the Project**: Double-click the `Master-Thesis.Rproj` file. This ensures R sets the working directory to the project root automatically.
2. **Run the Master Script**: Open `R/master.R` and click **Source**.

The `master.R` script will:
- Check for and install required R packages.
- Execute the full data cleaning and merging pipeline.
- Run OLS, Fixed Effects, and Instrumental Variable (IV) models.
- Export all descriptive and analytical plots to the `output/` folder.

## 🛠 Methodology Summary

- **Identification**: Superstar cities are identified through price-to-quantity growth ratios.
- **Robustness**: The analysis is performed both including and excluding the COVID-19 period (2020-2022).
- **Causal Strategy**: Uses WWII destruction shares as an instrument for modern housing supply inelasticity.

## Key Dependencies

- `tidyverse`: Data manipulation and visualization.
- `fixest`: High-dimensional fixed effects and IV estimation.
- `modelsummary`: Professional academic table generation.
- `patchwork`: Multi-panel figure composition.
